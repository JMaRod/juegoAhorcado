Juego del Ahorcado

Acerca del desarrollo del juego del ahorcado

Para cumplir el desafío se tenía que hacer un juego del ahorcado con las siguientes características

  • Se tenía que hacer con canvas.
  • El tablero de juego debía medir 1200x800 pixeles.
  • Se debían mostrar un mensaje para indicar que se había ganado o perdido el juego.
  • Se tenía que poner un botón para dar la opción para salir del juego
  • Tenía que poder añadir palabras nuevas al juego.
  • Solo debe trabajar con letras mayúsculas, sin espacios y sin caracteres especiales.

Mi propuesta para la solución del proyecto.

  • Se realizo un tablero con canvas, cumpliendo las medidas exactas que se pedía.
  • Para el fondo del canvas se "pinto" con el mismo canvas un fondo para simular una hoja de cuaderno.
  • Como página principal se puso un menú donde se pudiera escoger entre las opciones:
    • Jugar: donde te llevara al juego en si.
    • Instrucciones: te lleva a un apartado para conocer las instrucciones del juego.
    • Añadir palabra: donde se podrá añadir palabra.
    • Acerca de este proyecto: donde se da una explicación del proyecto y del porque se hizo

Algunos añádidos para la solucion

  • Dentro del juego se optó por poner los botones de las letras en el mismo tablero para de esta manera tener un mejor control de las letras que se usan para describir la palabra.
  • Se decidió incluir un pista junto con las palabras para ayudar al usuario final.
  • Se decidió añadir un letreo para advertir al usuario que solo queda un intento para terminar el juego.
  • Afuera del tablero se añadieron dos botones para regresar al inicio y para jugar otra palabra.
  • Para la sección de añadir palabras se decidió que se usarían dos forms para pedir la nueva palabra y su pista correspondiente; además de que se agregaron métodos para validar que la palabra no tenga espacios, minúsculas o números. Y en caso de encontrar los errores señalarlo en la página.
  • Se coloco una opción para revisar si hay un diccionario precargado y cuantas palabras hay en el mismo.
